# 🚀 Portfolio Developer — GitHub API

Website portfolio developer premium yang seluruh datanya diambil otomatis dari GitHub.

## ✨ Fitur
- Data otomatis dari GitHub API (profil, repo, bahasa, statistik)
- Desain dark mode modern (Glassmorphism)
- Responsif — mobile, tablet, desktop
- Filter, sort, dan pencarian project realtime
- Contribution heatmap
- Command Palette (CTRL+K)
- Terminal Mode rahasia (/terminal)
- PWA — installable & offline

## 🚀 Cara Deploy

1. Ganti `USERNAME` di `js/config.js`
2. Upload ke GitHub
3. Aktifkan GitHub Pages di Settings → Pages → Branch: main
